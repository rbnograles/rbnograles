## ðŸ“ˆ Customization Parameters Reference

```markdown
Available Parameters:
- layout: compact | donut | donut-vertical | pie
- theme: tokyonight | radical | gruvbox | dark | default | and many more
- langs_count: number of languages to show (default: 5)
- hide: comma-separated list of languages to hide
- exclude_repo: comma-separated list of repos to exclude
- card_width: custom width for the card
- hide_border: true/false
- bg_color: custom background color (hex without #)
```
